# IP Addressing, Binary & Hex - Notes

This file covers IPv4/IPv6 addressing and number system conversions.

---

## IPv4

- 32‑bit address.
- Written as four decimal numbers.
- Example: `192.168.1.1`

---

## IPv6

- 128‑bit address.
- Supports huge number of devices.
- Example: `2001:0db8:85a3::8a2e:0370:7334`

---

## Binary (Base‑2)

Used by computers internally.

Examples:

- 129 → `1000 0001`
- 78 → `0100 1110`
- 54 → `0011 0110`

---

## Hexadecimal (Base‑16)

Used to simplify binary.

Examples:

- 8193 → `2001`
- 3512 → `DB8`
- 61697 → `F101`

---

## Why It Matters

- Essential for subnetting.
- Used in MAC addresses, IPv6, memory, and low‑level networking.
