# TCP/IP Model - Notes

The TCP/IP model is the foundation of the internet. It uses **four layers** to define how data is transmitted across networks.

---

## TCP/IP Model Overview

| Layer | Purpose | Protocols |
|-------|----------|-----------|
| Application | Services for applications | HTTP, FTP, SMTP |
| Transport | Reliable delivery | TCP, UDP |
| Internet | Routing & addressing | IP, ICMP, IGMP |
| Network Access | Physical transmission | Ethernet, Wi‑Fi, ARP |

---

## Key Points

- Simpler than OSI (4 layers instead of 7).
- Real‑world networks use TCP/IP, not OSI.
- TCP ensures reliability; UDP is faster but not guaranteed.

---

## Security Relevance

- HTTPS uses TLS over TCP.
- IPsec protects data at the Internet layer.
- Understanding TCP/UDP helps detect scanning, spoofing, and DoS attacks.
