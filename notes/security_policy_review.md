# Security Policy Review - Notes

This file summarises key findings from reviewing the MidTown IT Security Policy.

---

## Visitor & External Personnel Access

**Issue:** Not included in the policy.  
**Why it matters:** Prevents unauthorised access and supports auditing.

---

## Social Media Usage

Improvements:

- Define acceptable use.
- Add training.
- Add incident reporting.
- Clarify personal vs professional use.
- Outline consequences.

---

## Official‑Sensitive Data Access

People with access:

- Authorised employees  
- IT admins  
- Backup personnel  

Controls:

- Encryption  
- Auditing  
- Classification labels  

Improvements:

- MFA  
- Access reviews  
- Incident response plan  
- Zero‑trust model  

---

## BYOD Risks

Risks:

- Malware  
- Data leakage  
- Compliance issues  
- Lost/stolen devices  

Improvements:

- Device registration  
- Remote wipe  
- Minimum security requirements  
- Training  
- Regular audits
