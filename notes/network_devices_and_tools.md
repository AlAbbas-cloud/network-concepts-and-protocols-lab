# Network Devices & Tools - Notes

This file summarises key networking devices and testing tools used in cyber security.

---

## Network Devices

### **Router**
- Connects networks.
- Performs routing and NAT.

### **Switch**
- Connects devices within a LAN.
- Uses MAC addresses.

### **Firewall**
- Filters traffic.
- Enforces security rules.

### **Access Point**
- Provides wireless connectivity.

---

## Testing Tools

### **Wireshark**
- Packet capture and analysis.

### **Nmap**
- Port scanning and network discovery.

### **Packet Tracer**
- Network simulation and design.

### **Ping / Traceroute**
- Connectivity and path testing.

---

## Why It Matters

Understanding devices and tools is essential for:

- Troubleshooting  
- Network design  
- Security testing  
- SOC analysis
